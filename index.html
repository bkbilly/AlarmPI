LNX Link

Home Assistant Companion APP

  • Github

About

LNX Link is a Linux service for integrating your system with an external application using MQTT.

It starts automatically on boot and you don't need to be a root user to install and use.

Features

System control: Shutdown, Restart, Send Keys, Notify, Media.

System monitor: CPU, Ram, Network, Media.

Easily expanded: Modular design so that programmers can add more features just by creating a new application.

Install

To Install or Update LNX Link, open a terminal and enter:

bash <(curl -s "https://raw.githubusercontent.com/bkbilly/lnxlink/master/install.sh")

Follow the on-screen instructions to complete the installation.

Home Assistant integration

Using Autoconfig, Home Assistant can find all the sensors and switches automatically.

Using this custom integration you can have a media player to control your PC on your home assistant frontend.
